CSS internos

Acredito que dentro da metodologia DRY (DON'T REPEAT YUORSELF) o uso das CSS inline nao deve ser aplicado, uma vez que exige que se repita as mesmas intrucoes vez-apos-vez para Tags similares A melhor pratica eh colocar o CSS interno, ou local.

Quando usar?

Quando houver necessidade de especificar ou diferenciar uma TAG especifica a CSS inline seria aplicavel
